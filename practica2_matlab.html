🎯 Objetivo

Simular el comportamiento de un sistema masa-resorte amortiguado usando MATLAB y analizar el efecto del coeficiente de amortiguamiento en la respuesta del sistema.

Contexto agroindustrial: vibración en bandas transportadoras y equipos de agitación.

⚙️ Modelo matemático
\[ m\,x'' + c\,x' + k\,x = F(t) \]

Parámetros base: \( m = 2\;\text{kg} \), \( k = 8\;\text{N/m} \), \( x(0)=1\;\text{m} \), \( x'(0)=0 \). Variación de \( c \).

\[ \text{Raíz característica: }\lambda = \frac{-c \pm \sqrt{c^2 - 4mk}}{2m} \]
💻 Actividades
  • Convierte la EDO de orden 2 al sistema de primer orden: \( x_1 = x \), \( x_2 = x' \) y resuelve con ode45
  • Simula con 5 valores distintos de c: c = 0, 2, 4, 8, 16 N·s/m
  • Para cada caso clasifica el sistema: subamortiguado, críticamente amortiguado o sobreamortiguado
  • Grafica las 5 respuestas en una sola figura con leyenda y colores distintos
  • Agrega una segunda figura con el plano fase \( (x,\,x') \) para c = 0 y c = 4
  • Calcula analíticamente el valor crítico \( c_{cr} = 2\sqrt{mk} \) y verifícalo en la simulación
Estructura del sistema de primer orden

f = @(t,X) [X(2); (-c*X(2) - k*X(1))/m];
[t,X] = ode45(f, [0 10], [1; 0]);
x = X(:,1); % posición
v = X(:,2); % velocidad

📄 Documentación y diagrama de flujo

El script principal debe incluir encabezado documentado. El reporte PDF debe incluir el diagrama de flujo del algoritmo mostrando: inicialización, ciclo para cada valor de c, llamada a ode45, clasificación del sistema y generación de gráficas.

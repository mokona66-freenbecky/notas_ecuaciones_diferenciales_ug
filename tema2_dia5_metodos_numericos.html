🧪 Práctica 1 — Métodos Numéricos

En esta sesión implementarás los principales métodos numéricos para resolver EDOs de primer orden desde cero en MATLAB, los compararás con la solución analítica y con el solver ode45 de MATLAB para evaluar su precisión y costo computacional.

Método 1

Euler Explícito

El más simple. Avanza en la dirección de la pendiente actual. Error local proporcional a h².

Orden 1 — Error O(h)
Método 2

Runge-Kutta 4 (RK4)

Promedia 4 evaluaciones de la pendiente. Mucho más preciso que Euler con el mismo h.

Orden 4 — Error O(h⁴)
Método 3

MATLAB ode45

Solver adaptativo RK45. Ajusta h automáticamente según la tolerancia. El estándar industrial.

Adaptativo — Referencia

🟢 Método de Euler Explícito

// Dado: y' = f(t,y), y(t₀) = y₀
// Paso h, n pasos hasta t_final

\(y_{n+1} = y_n + h\,f(t_n,\,y_n)\)

tₙ₊₁ = tₙ + h
// Error de truncamiento local: O(h²)
// Error global: O(h) — orden 1
euler_explicito.m — implementar desde cero
function [t, y] = euler_explicito(f, t0, tf, y0, h)
% EULER_EXPLICITO — Método de Euler de orden 1
% Entradas: f = @(t,y) función, t0 y tf = intervalo
% y0 = condición inicial, h = tamaño de paso

t = t0:h:tf;
n = length(t);
y = zeros(1, n);
y(1) = y0;

for i = 1:n-1
  y(i+1) = y(i) + h * f(t(i), y(i));
end
end

🔵 Método Runge-Kutta de Orden 4 (RK4)

\(k_1 = h\,f(t_n,\;y_n)\)
\(k_2 = h\,f\!\left(t_n+\tfrac{h}{2},\;y_n+\tfrac{k_1}{2}\right)\)
\(k_3 = h\,f\!\left(t_n+\tfrac{h}{2},\;y_n+\tfrac{k_2}{2}\right)\)
\(k_4 = h\,f(t_n+h,\;y_n+k_3)\)

\(y_{n+1} = y_n + \dfrac{k_1 + 2k_2 + 2k_3 + k_4}{6}\)

// Error de truncamiento local: O(h⁵)
// Error global: O(h⁴) — orden 4
rk4.m — implementar desde cero
function [t, y] = rk4(f, t0, tf, y0, h)
% RK4 — Runge-Kutta de orden 4
t = t0:h:tf;
n = length(t);
y = zeros(1, n);
y(1) = y0;

for i = 1:n-1
  k1 = h * f(t(i), y(i));
  k2 = h * f(t(i) + h/2, y(i) + k1/2);
  k3 = h * f(t(i) + h/2, y(i) + k2/2);
  k4 = h * f(t(i) + h, y(i) + k3);
  y(i+1) = y(i) + (k1 + 2*k2 + 2*k3 + k4) / 6;
end
end
🏭 Problema de Aplicación Agroindustrial

Enfriamiento de leche pasteurizada (Ley de Newton)

Un lote de leche sale del pasteurizador a T₀ = 85°C. La leche se enfría en un ambiente a T∞ = 4°C. La constante de enfriamiento es k = 0.0369 min⁻¹. La EDO que modela el proceso es:

\[ \frac{dT}{dt} = -0.0369\,(T - 4), \qquad T(0) = 85^\circ\text{C} \] \[ T_{\text{exacta}}(t) = 4 + 81\,e^{-0.0369\,t} \]
T₀
85°C
T∞
4°C
k
0.0369 min⁻¹
t_final
130 min
Meta T
10°C

🟤 Código Completo de Comparación

comparacion_metodos.m — código principal de la práctica
%% PRÁCTICA 1 — Métodos Numéricos para EDO
%% Dra. María de la Luz López González — Ecuaciones Diferenciales
%% Problema: Enfriamiento de leche pasteurizada

clear; clc; close all

%% 1. Definición del problema
f = @(t,T) -0.0369 * (T - 4); % EDO: dT/dt = -k(T-T_inf)
t0 = 0; tf = 130; T0 = 85;
T_exacta = @(t) 4 + 81*exp(-0.0369*t);

%% 2. Resolver con tres valores de h
h_vals = [10, 5, 1]; % tamaños de paso a comparar
colores = {'r--', 'm-.', 'g:'};

figure(1); hold on;
t_ref = 0:0.1:tf;
plot(t_ref, T_exacta(t_ref), 'k-', 'LineWidth', 2.5, 'DisplayName', 'Exacta')

for j = 1:length(h_vals)
  h = h_vals(j);
  [t_eu, T_eu] = euler_explicito(f, t0, tf, T0, h);
  [t_rk, T_rk] = rk4(f, t0, tf, T0, h);
  plot(t_eu, T_eu, colores{j}, 'DisplayName', ['Euler h=' num2str(h)])
  plot(t_rk, T_rk, 'b', 'DisplayName', ['RK4 h=' num2str(h)])
end

%% ode45 (referencia adaptativa)
[t_od, T_od] = ode45(f, [t0 tf], T0);
plot(t_od, T_od, 'c-s', 'LineWidth', 1.5, 'DisplayName', 'ode45')

yline(10, 'k--', 'Meta: 10°C')
xlabel('Tiempo (min)'); ylabel('Temperatura (°C)')
title('Comparación de métodos numéricos — Enfriamiento de leche')
legend('Location', 'northeast'); grid on

%% 3. Análisis de error global
figure(2)
h_test = [20, 10, 5, 2, 1, 0.5];
err_euler = zeros(size(h_test)); err_rk4 = err_euler;

for j = 1:length(h_test)
  h = h_test(j);
  [~, yE] = euler_explicito(f, t0, tf, T0, h);
  [~, yR] = rk4(f, t0, tf, T0, h);
  T_ex = T_exacta(t0:h:tf);
  err_euler(j) = max(abs(yE - T_ex));
  err_rk4(j) = max(abs(yR - T_ex));
end

loglog(h_test, err_euler, 'r-o', 'LineWidth', 2, 'DisplayName', 'Euler')
hold on
loglog(h_test, err_rk4, 'b-s', 'LineWidth', 2, 'DisplayName', 'RK4')
xlabel('Tamaño de paso h'); ylabel('Error máximo global')
title('Error global vs tamaño de paso (escala log-log)')
legend; grid on

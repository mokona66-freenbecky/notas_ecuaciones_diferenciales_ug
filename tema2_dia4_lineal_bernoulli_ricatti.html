Subtema 2.5

EDO Lineal de 1er Orden

y' + P(x)y = Q(x). Se resuelve con factor integrante μ = e^∫P(x)dx. Clave: la solución siempre existe si P y Q son continuas.

Subtema 2.6a

Ecuación de Bernoulli

\(y' + P(x)\,y = Q(x)\,y^n, \quad n\neq 0,1\). La sustitución \(v = y^{1-n}\) la convierte en una EDO lineal en v. Aplica para n ≠ 0,1.

Subtema 2.6b

Ecuación de Ricatti

y' = P(x) + Q(x)y + R(x)y². Requiere conocer una solución particular y₁. La sustitución \(y = y_1 + \dfrac{1}{v}\) la reduce a lineal.

🔵 2.5 EDO Lineal de Primer Orden

Forma estándar
\(y' + P(x)\,y = Q(x)\)
Factor integrante: μ(x) = e^∫P(x)dx  →  Solución: \(y = \dfrac{1}{\mu}\left[\int \mu\,Q\,dx + C\right]\)
  • 1
    Escribir en forma estándar: y' + P(x)y = Q(x)
  • 2
    Calcular factor integrante: \(\mu(x) = e^{\int P(x)\,dx}\)
  • 3
    Multiplicar ambos lados: μy' + μPy = μQ → (μy)' = μQ
  • 4
    Integrar: μy = ∫μQ dx + C
  • 5
    Despejar y: y = (1/μ)[∫μQ dx + C]
Ejemplo — Mezcla en tanque de salmuera
EDO:\(\dfrac{dS}{dt} + \dfrac{1}{50}S = 6\) \((S\) en kg, \(t\) en min\()\)
P(t):P = 2/100 = 1/50
μ:μ = e^(t/50)
(μS)':(e^(t/50)·S)' = 6e^(t/50)
Integrar:e^(t/50)·S = 300e^(t/50) + C
Solución:\(S(t) = 300 + Ce^{-t/50}\)
S(0)=50:C = -250 → \(S(t) = 300 - 250\,e^{-t/50}\)
MATLAB — edo_lineal.m
% EDO lineal — mezcla de salmuera
syms S(t)
ode = diff(S,t) + (1/50)*S == 6;
sol = dsolve(ode, S(0)==50)

% Graficar y analizar estado estacionario
S_fun = matlabFunction(sol);
t_vec = 0:1:300;
plot(t_vec, S_fun(t_vec), 'b-', 'LineWidth', 2); hold on
yline(300, 'r--', 'Estado estacionario = 300 kg')
xlabel('Tiempo (min)'); ylabel('Sal (kg)')
title('Mezcla en tanque de salmuera'); grid on

🟣 2.6a Ecuación de Bernoulli

Forma de Bernoulli
\(y' + P(x)\,y = Q(x)\)·yⁿ   (n ≠ 0, 1)
Sustitución: \(v = y^{1-n}\)  →  v' + (1-n)P(x)v = (1-n)Q(x)   [lineal en v]
  • 1
    Identificar n del término yⁿ al lado derecho
  • 2
    Dividir toda la EDO entre yⁿ
  • 3
    Hacer la sustitución \(v = y^{1-n}\), entonces v' = (1-n)y^(-n)y'
  • 4
    Reescribir en términos de v → queda una EDO lineal en v
  • 5
    Resolver la EDO lineal en v, luego regresar: y = v^(1/(1-n))
Ejemplo — Modelo logístico de crecimiento microbiano
EDO:\(\dfrac{dN}{dt} = rN - \dfrac{r}{K}N^2\)
n = 2:\(v = N^{-1}\)
v':v' = -N^(-2)N' → N' = -N²v'
Lineal en v:v' + rv = r/K
Solución v:v(t) = 1/K + Ce^(-rt)
N(t):\(N(t) = \dfrac{K}{1+(K/N_0-1)e^{-rt}}\quad\leftarrow\) Modelo logístico
MATLAB — bernoulli_logistico.m
% Modelo logístico (Bernoulli) — Crecimiento de levaduras
r = 0.5; K = 1000; N0 = 50;
N = @(t) K ./ (1 + (K/N0 - 1)*exp(-r*t));

t_vec = 0:0.1:20;
plot(t_vec, N(t_vec), 'm-', 'LineWidth', 2); hold on
yline(K, 'k--', 'Capacidad de carga K')
xlabel('Tiempo (h)'); ylabel('N(t) — células/mL')
title('Crecimiento logístico de levaduras (fermentación)')
grid on; legend('N(t)', 'K = 1000')

🟤 2.6b Ecuación de Ricatti

Forma de Ricatti
y' = P(x) + Q(x)·y + R(x)·y²
Requiere solución particular y₁. Sustitución: \(y = y_1 + \dfrac{1}{v}\)  →  v' + [Q + 2Ry₁]v = -R   [lineal]
  • 1
    Identificar P(x), Q(x), R(x) de la EDO de Ricatti
  • 2
    Encontrar (o recibir) una solución particular y₁ por inspección o prueba
  • 3
    Sustituir \(y = y_1 + \dfrac{1}{v}\), calcular y' = y₁' - v'/v²
  • 4
    Sustituir en la EDO → se obtiene una EDO lineal en v
  • 5
    Resolver v(x) y regresar: \(y = y_1 + \dfrac{1}{v}\)
Ejemplo — Modelo de difusión con reacción
EDO:y' = 1 - y + y²   (P=1, Q=-1, R=1)
Sol. part.:y₁ = 1 (verificar: 1' = 0 = 1-1+1 ✗ → probar y₁=1: 0 ≠ 1 ✗, probar y₁: 0 = 1-y₁+y₁², y₁ = (1±√5)/2)
y₁ = (1+√5)/2 ≈ 1.618 (número áureo)
Sust.:\(y = y_1 + \dfrac{1}{v}\) → v' + (-1 + 2y₁)v = -1 → EDO lineal
Solución:y(x) = y₁ + 1/v(x) (completar con factor integrante)
MATLAB — comparacion_analitica_numerica.m
% Comparación: solución analítica (dsolve) vs numérica (ode45)
% EDO lineal: y' + (1/50)y = 6, y(0) = 50
syms y_s(t)
sol_analitica = dsolve(diff(y_s,t) + (1/50)*y_s == 6, y_s(0)==50);

% Solución numérica con ode45
f = @(t,y) -(1/50)*y + 6;
[t_num, y_num] = ode45(f, [0 300], 50);

% Comparar
t_vec = 0:1:300;
plot(t_vec, subs(sol_analitica, t, t_vec), 'b-', 'LineWidth', 2); hold on
plot(t_num, y_num, 'r--', 'LineWidth', 1.5)
legend('Analítica (dsolve)', 'Numérica (ode45)')
title('Comparación: Analítica vs Numérica'); grid on

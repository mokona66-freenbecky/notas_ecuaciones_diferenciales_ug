🎯 Objetivo

Aplicar la Transformada de Laplace en MATLAB usando el Symbolic Toolbox para resolver EDO con condiciones iniciales y comparar con la solución numérica.

🧮 Fundamento teórico
\[ \mathcal{L}\{y'\} = s\,Y(s) - y(0) \qquad \mathcal{L}\{y''\} = s^2 Y(s) - s\,y(0) - y'(0) \]
\[ Y(s) = \mathcal{L}\{y(t)\} \qquad y(t) = \mathcal{L}^{-1}\{Y(s)\} \]
💻 Actividades

Resuelve los siguientes PVI usando laplace e ilaplace del Symbolic Toolbox:

a)\( y'' + 4y' + 4y = e^{-2t},\quad y(0)=0,\;y'(0)=1 \)
b)\( y'' + 2y' + 5y = \sin(t),\quad y(0)=1,\;y'(0)=0 \)
c)\( y'' - y = t\,e^t,\quad y(0)=0,\;y'(0)=0 \)

Para cada inciso:

  • Aplica laplace a la EDO y despeja Y(s) simbólicamente
  • Aplica ilaplace para obtener y(t)
  • Verifica con dsolve y con ode45
  • Grafica las tres soluciones (Laplace, dsolve, ode45) en una sola figura
  • Calcula y reporta el error máximo entre la solución de Laplace y ode45
Estructura básica en MATLAB

syms y(t) s Y
ode = diff(y,t,2) + 4*diff(y,t) + 4*y == exp(-2*t);
% Aplicar Laplace manualmente o con laplace()
Y_s = laplace(rhs(ode), t, s);
y_t = ilaplace(Y_s, s, t);

Subtema 2.3

Ecuación Exacta

M(x,y)dx + N(x,y)dy = 0 es exacta si \(\dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}\). Existe una función potencial F tal que dF = 0, con ∂F/∂x = M y ∂F/∂y = N.

Subtema 2.4

Factor Integrante

Si la EDO no es exacta, se multiplica por un factor μ(x) o μ(y) que la vuelve exacta. Se determina a partir de (∂M/∂y - ∂N/∂x)/N o (∂N/∂x - ∂M/∂y)/M.

🟠 2.3 Ecuación Exacta

⚠️ Condición de Exactitud
M(x,y)dx + N(x,y)dy = 0 es exacta si y solo si:
\(\dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}\)   en toda región R
// Si es exacta, existe F(x,y) tal que:
\(\dfrac{\partial F}{\partial x} = M(x,y)\)   y   \(\dfrac{\partial F}{\partial y} = N(x,y)\)

// Solución implícita:
\(F(x,y) = C\)
  • 1
    Identificar M(x,y) y N(x,y) de la ecuación M dx + N dy = 0
  • 2
    Calcular ∂M/∂y y ∂N/∂x. Si son iguales → la ecuación es exacta
  • 3
    Integrar M respecto a x: F = ∫M dx + g(y)
  • 4
    Derivar F respecto a y e igualar a N para encontrar g'(y)
  • 5
    Integrar g'(y) para obtener g(y). Escribir la solución \(F(x,y) = C\)
Ejemplo resuelto — Balance de masa en proceso de mezcla
EDO:(2xy + 3)dx + (x² - 4y)dy = 0
M, N:M = 2xy+3   N = x²-4y
Exacta:∂M/∂y = 2x  =  ∂N/∂x = 2x ✓
F = ∫M dx:F = x²y + 3x + g(y)
∂F/∂y = N:x² + g'(y) = x²-4y → g'(y) = -4y
g(y):g(y) = -2y²
Solución:F = x²y + 3x - 2y² = C
MATLAB — exacta.m
% Verificar exactitud y resolver simbólicamente
syms x y
M = 2*x*y + 3;
N = x^2 - 4*y;

dMdy = diff(M, y);
dNdx = diff(N, x);
disp('∂M/∂y = '); disp(dMdy)
disp('∂N/∂x = '); disp(dNdx)

% Calcular función potencial F
F = int(M, x) + int(N - diff(int(M,x), y), y);
disp('F(x,y) = '); disp(simplify(F))

% Graficar curvas de nivel F = C
f = @(x,y) x.^2.*y + 3*x - 2*y.^2;
[X,Y] = meshgrid(-3:0.1:3);
contour(X, Y, f(X,Y), 20); colorbar
title('Curvas de nivel \(F(x,y) = C\)'); grid on

🔵 2.4 Factor Integrante

Cuando la EDO NO es exacta
Se multiplica por μ para que μM dx + μN dy = 0 sea exacta.

μ = μ(x) cuando:

\(\dfrac{\partial M/\partial y - \partial N/\partial x}{N} = f(x)\)
solo depende de x

Entonces: \(\mu(x) = e^{\int f(x)\,dx}\)

μ = μ(y) cuando:

\(\dfrac{\partial N/\partial x - \partial M/\partial y}{M} = g(y)\)
solo depende de y

Entonces: \(\mu(y) = e^{\int g(y)\,dy}\)

  • 1
    Verificar que la EDO NO sea exacta: ∂M/∂y ≠ ∂N/∂x
  • 2
    Calcular (∂M/∂y - ∂N/∂x)/N. Si solo depende de x → μ = μ(x)
  • 3
    Calcular (∂N/∂x - ∂M/∂y)/M. Si solo depende de y → μ = μ(y)
  • 4
    Calcular μ integrando y multiplicar toda la EDO por μ
  • 5
    Resolver la EDO exacta resultante con el método anterior
Ejemplo resuelto — Cinética enzimática en bioreactor
EDO:(xy - 1)dx + (x² - xy)dy = 0   (no exacta)
M, N:M = xy-1  |  N = x²-xy
∂M/∂y:x  ≠  ∂N/∂x = 2x-y → no exacta
(∂M/∂y - ∂N/∂x)/N:(x - (2x-y))/(x²-xy) = (-x+y)/(x(x-y)) = -1/x
μ(x):e^∫(-1/x)dx = e^(-ln x) = 1/x
Nueva EDO:(y - 1/x)dx + (x - y)dy = 0 → exacta ✓
Solución:xy - ln|x| - y²/2 = C
MATLAB — factor_integrante.m
% Verificar exactitud y calcular factor integrante
syms x y
M = x*y - 1;
N = x^2 - x*y;

expr = (diff(M,y) - diff(N,x)) / N;
disp('(∂M/∂y - ∂N/∂x)/N = '); disp(simplify(expr))

% Factor integrante \(\mu(x) = e^{\int f(x)\,dx}\)
f_x = -1/x;
mu = exp(int(f_x, x));
disp('Factor integrante μ(x) = '); disp(mu)

% Multiplicar y resolver EDO exacta resultante
M2 = simplify(mu * M);
N2 = simplify(mu * N);
F = int(M2, x) + int(N2 - diff(int(M2,x),y), y);
disp('Solución \(F(x,y) = C\):'); disp(simplify(F))

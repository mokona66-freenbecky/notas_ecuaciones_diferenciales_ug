🎯 Objetivo

Resolver la ecuación de Laplace 2D mediante diferencias finitas en Python y visualizar la distribución de temperatura en una placa rectangular — modelo de pared de cámara frigorífica.

🌡️ Modelo matemático
\[ \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0, \qquad (x,y)\in[0,L_x]\times[0,L_y] \]

Condiciones de frontera:

\[ u(0,y)=0, \quad u(L_x,y)=0, \quad u(x,0)=0, \quad u(x,L_y)=\sin\!\left(\frac{\pi x}{L_x}\right) \cdot 100 \]

Parámetros: \( L_x = L_y = 1\;\text{m} \), malla de 50×50 puntos

🐍 Actividades en Python

Esquema de diferencias finitas centradas:

\[ u_{i,j} = \frac{u_{i+1,j} + u_{i-1,j} + u_{i,j+1} + u_{i,j-1}}{4} \]
  • Implementa el método iterativo de Gauss-Seidel o Jacobi para resolver el sistema
  • Define criterio de convergencia: \( \max|u^{k+1} - u^k| < 10^{-6} \)
  • Genera las siguientes visualizaciones con matplotlib:
    • Mapa de calor 2D con imshow o pcolormesh y barra de color
    • Superficie 3D con plot_surface de mpl_toolkits.mplot3d
    • Líneas isotérmicas con contour
  • Grafica la convergencia del error máximo vs número de iteraciones
  • Compara el resultado con la solución analítica: \( u = \sin(\pi x)\sinh(\pi y)/\sinh(\pi) \cdot 100 \)
  • Repite con malla de 25×25 y 100×100 y compara tiempos de cálculo

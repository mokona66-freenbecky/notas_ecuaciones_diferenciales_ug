📐 1.2 Interpretación Geométrica — Campos de Dirección

Un campo de dirección asigna a cada punto (x, y) una pendiente m = f(x, y) dada por la EDO. Las curvas solución son tangentes al campo en cada punto.

EDO de primer orden — campo de dirección
\[ \frac{dy}{dx} = f(x,\,y) \] \[ \text{Isoclinas: } f(x,y) = c \quad \Rightarrow \quad \text{pendiente constante } c \]
MATLAB — campo_direccion.m
% Campo de dirección para dy/dx = y - x
% Aplicación: modelo simplificado de temperatura en proceso
[x, y] = meshgrid(-3:0.4:3, -3:0.4:3);
dydx = y - x;

quiver(x, y, ones(size(x)), dydx, 0.5)
xlabel('x'); ylabel('y')
title('Campo de dirección: dy/dx = y - x')
axis equal; grid on

🏭 1.3 Problemas Físicos que dan lugar a EDO

🧫
Crecimiento microbiano dN/dt = kN  →  modelo exponencial
🌡️
Enfriamiento de leche dT/dt = -k(T - T∞)  →  Ley de Newton
🧪
Mezcla en tanque dQ/dt = entrada - salida  →  balance de masa
💧
Drenaje de silo dh/dt = -c√h  →  Ley de Torricelli
⚗️
Reacción de primer orden dC/dt = -kC  →  cinética química
📦
Secado de granos dM/dt = -k(M - Me)  →  modelo de Lewis

📋 1.4 Teorema de Existencia y Unicidad de Soluciones

Teorema de Picard (Existencia y Unicidad)

Sea f(x, y) continua en una región rectangular R que contiene al punto (x₀, y₀). Si ∂f/∂y existe y es continua en R, entonces existe un intervalo (x₀ - h, x₀ + h) en el que el PVI y' = f(x, y), y(x₀) = y₀ tiene una única solución.

Problema de Valor Inicial (PVI)
y' = f(x, y) , y(x₀) = y₀

Condiciones: f continua en R y ∂f/∂y continua en R
→ ∃! solución en algún intervalo alrededor de x₀
MATLAB — verificacion_unicidad.m
% Verificar unicidad resolviendo un PVI con dsolve
syms y(x)
ode = diff(y,x) == y - x;
cond = y(0) == 2;
sol = dsolve(ode, cond);
disp('Solución única:'); disp(sol)

% Graficar la solución particular
fplot(matlabFunction(sol), [-2, 4], 'LineWidth', 2)
grid on; xlabel('x'); ylabel('y(x)')
title('Solución del PVI: y''=y-x, y(0)=2')

Subtema 2.1

Variables Separables

La EDO puede escribirse como g(y)dy = f(x)dx. Se integran ambos lados de forma independiente. Es el método más directo cuando la separación es posible.

Subtema 2.2

Ecuación Homogénea

f(x,y) es homogénea de grado n si f(tx,ty) = tⁿ·f(x,y). La sustitución v = y/x la convierte en una ecuación de variables separables.

🟢 2.1 Método de Variables Separables

Método de Variables Separables
\[ \frac{dy}{dx} = f(x)\cdot g(y) \] \[ \frac{dy}{g(y)} = f(x)\,dx \] \[ \int \frac{dy}{g(y)} = \int f(x)\,dx + C \]
  • 1
    Verificar que la EDO sea separable: escribirla como dy/dx = f(x)·g(y)
  • 2
    Dividir entre g(y) y multiplicar por dx: dy/g(y) = f(x)dx
  • 3
    Integrar ambos lados: ∫dy/g(y) = ∫f(x)dx + C
  • 4
    Despejar y si es posible; incluir la constante C
  • 5
    Aplicar condición inicial si es un PVI para encontrar C
Ejemplo resuelto — Enfriamiento de leche pasteurizada
EDO:\(\dfrac{dT}{dt} = -k(T-4),\quad T(0)=85^\circ\text{C},\quad T(10)=60^\circ\text{C}\)
Separar:\(\dfrac{dT}{T-4} = -k\,dt\)
Integrar:\(\ln|T-4| = -kt + C_1\)
Solución:\(T(t) = 4 + 81\,e^{-kt}\)
Hallar k:\(k = -\dfrac{\ln(56/81)}{10} \approx 0.0369\;\text{min}^{-1}\)
¿T = 4°C?\(t = \dfrac{\ln(81)}{0.0369} \approx 118.5\;\text{min}\)
MATLAB — variables_separables.m
% Solución simbólica — variables separables
syms T(t) k
ode = diff(T,t) == -k*(T-4);
sol = dsolve(ode, T(0)==85)

% Con k numérico para graficar
k_val = 0.0369;
T_sol = @(t) 4 + 81*exp(-k_val*t);
t_vec = 0:1:130;
plot(t_vec, T_sol(t_vec), 'b-', 'LineWidth', 2)
yline(4, 'r--', 'T ambiente')
xlabel('Tiempo (min)'); ylabel('Temperatura (°C)')
title('Enfriamiento de leche — Ley de Newton'); grid on

🟣 2.2 Ecuación Homogénea

Ecuación Homogénea — Sustitución
\[ f(tx,\,ty) = t^n\,f(x,\,y) \] \[ M(x,y)\,dx + N(x,y)\,dy = 0, \quad M,N \text{ homogéneas del mismo grado} \] \[ v = \frac{y}{x} \;\Rightarrow\; y = vx,\quad dy = v\,dx + x\,dv \]
  • 1
    Verificar homogeneidad: comprobar que M y N son del mismo grado con f(tx,ty) = tⁿf(x,y)
  • 2
    Sustituir v = y/x, es decir y = vx y dy = v dx + x dv
  • 3
    Reescribir la EDO en términos de v y x — quedará separable
  • 4
    Separar variables y resolver la integral en v y x
  • 5
    Regresar a las variables originales: sustituir v = y/x
Ejemplo resuelto — Flujo en canal agroindustrial
EDO:\((x^2+y^2)\,dx - 2xy\,dy = 0\)
Grado:M y N son homogéneas de grado 2 ✓
Sustit.:\(y=vx,\quad dy = v\,dx + x\,dv\)
Separa:\(\dfrac{dx}{x} = \dfrac{2v\,dv}{v^2-1}\)
Integra:\(\ln|x| = \ln|v^2-1| + C\)
Regresa:\(x^3 = C(y^2 - x^2)\)
MATLAB — homogenea.m
% Resolución simbólica — ecuación homogénea
syms y(x)
ode = (diff(y,x)) == (x^2 + y^2)/(2*x*y);
sol = dsolve(ode)

% Graficar familia de soluciones
for C = [-2, -1, 0, 1, 2]
  fimplicit(@(x,y) x.^3 - C*(y.^2-x.^2), ...
    [-3 3 -3 3]); hold on
end
grid on; title('Familia de soluciones — Homogénea')

🔍 ¿Cómo identificar el método correcto?

✅ Variables Separables — úsalo cuando:

  • dy/dx se puede escribir como f(x)·g(y)
  • Al mover términos, x queda de un lado y y del otro
  • Ejemplo: dy/dx = x²y → separable

✅ Homogénea — úsala cuando:

  • M(x,y) y N(x,y) tienen el mismo grado homogéneo
  • La EDO se puede escribir como dy/dx = F(y/x)
  • Ejemplo: dy/dx = (x+y)/x → homogénea
